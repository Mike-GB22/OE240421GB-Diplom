BusinessLogicHandler

Ожидаем от клиента запрос типа:
@команда:аргументы (разделеные при необходимости знаком :)


Команда (аргументы), регистр не важен :
 * Автораизация
login       (user_id, pass) - вход по логину и паролю
exit        - выход, удаление сессии
session     (SID) - вход по уже созданной сессии

 * Отправка сообщений (тема и сообщение разденны знаком %%%:%%%):
msg        (topic, msg)
private    (user_id, topic, msg)
edit       (messageId, topic, msg)
del        (messageId)

 * Стриминг получение сообщений (клиент открывает сессию, и получает сообщения даже без запроса)
streamOn    - включить в список
streamOff   - изключить из спика

 * Сообщения по запросу
get100      - получить последние 100 сообщений
getLast    (index) - получить послдение сообщения с index
getBetween (index, index) - получить послдение сообщения с index по индекс
